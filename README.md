# Generative AI for Electronic Component Description Generation

This project uses a transformer-based Generative AI model to generate descriptions for electronic components such as resistors, capacitors, diodes, and transistors.

## Objective

To generate technical descriptions of electronic components automatically using AI.

## Input

Electronic component name

## Output

Generated technical description of the component

## Technologies Used

* Python
* HuggingFace Transformers
* GPT-2

## Example

Input: Resistor
Output: A resistor limits the flow of electric current in a circuit.
